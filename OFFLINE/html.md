# HTML & CSS
### emmit
`#green>li*2+li>ol>li*2`
```html
<ul id="green">
    <li>알고리즘</li>
    <li>웹</li>
    <li>프론트
        <ol>
            <li>HTML</li>
            <li>CSS</li>
        </ol>
    </li>
</ul>
```

### Style 우선순위
| 순위 | 보기            |
| ------ | --------------- |
| 1 | `!important` |
| 2 | `Inline style` |
| 3 | `selector` (id > class) |
| 4 | 스타일 코드 `순서` |
| 5 | 일반 태그 `h1`, `p` |
